# Jiuge 更新日志

## [2.0.0] - 开发中

### 新增
- 🍎 Apple Music 适配器（MusicKit API）
- 🔄 跨平台歌单同步服务
- 📁 本地音乐库扫描和上传
- 🧠 智能推荐引擎（协同过滤 + 内容推荐）
- 🌐 Web API 服务（FastAPI）
- 📡 WebSocket 实时状态推送

- 更完善的歌曲匹配算法（ISRC + 元数据）
- 播放历史时间衰减因子
- 异步性能优化
