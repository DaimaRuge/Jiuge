## 2. 跨平台歌单同步

### 2.1 歌曲匹配算法

```python
def match_song(source_track: Track, target_platform: str) -> Optional[str]:
    """
    匹配策略（按优先级）：
    1. ISRC 国际标准录音代码
    2. 歌曲名 + 主唱歌手 + 时长（±5秒）
    3. 歌曲名 + 歌手名（模糊匹配）
    """
    # 1. ISRC 匹配
    if source_track.isrc:
        result = search_by_isrc(source_track.isrc, target_platform)
        if result:
            return result
    
    # 2. 元数据匹配
    query = f"{source_track.title} {source_track.artists[0]}"
    results = search(query, target_platform)
    
    for track in results:
        if is_similar(source_track, track):
            return track.id
    
    return None
```

### 2.2 同步流程

```
1. 获取源歌单所有歌曲

3. 创建/更新目标歌单

import mutagen

def parse_metadata(file_path: str) -> dict:
    """解析音频文件元数据"""
    audio = mutagen.File(file_path)
    
    return {
        "title": audio.get("title", [""])[0],
        "artist": audio.get("artist", [""])[0],
        "album": audio.get("album", [""])[0],
        "duration": int(audio.info.length),
        "bitrate": audio.info.bitrate,
        "sample_rate": getattr(audio.info, "sample_rate", None)
    }
```

### 4.3 时间衰减

```python
def time_decay_score(play_time: datetime, half_life: int = 30) -> float:
    """
    时间衰减因子
    
    half_life: 半衰期（天）
    """
    days_ago = (datetime.now() - play_time).days
    return 0.5 ** (days_ago / half_life)
```
